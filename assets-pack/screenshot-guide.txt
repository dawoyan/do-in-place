DO IN PLACE — SCREENSHOT GUIDE (Google Play)
═══════════════════════════════════════════════════════

Google Play requires at minimum 2 screenshots, up to 8.
Phone screenshots: 16:9 or 9:16. Recommended: 1080×1920 px.

Shoot on a physical device or emulator. All screenshots are portrait.
Add overlay text using Figma, Canva, or any design tool.

Overlay style guide:
  Background bar (top or bottom): #0A1628 at 85% opacity
  Headline: White, bold, 36–44 pt
  Subtext: #10B981 teal or #94A3B8 slate, regular, 22 pt
  Font: any clean sans-serif (match the app's Material 3 look)


── SCREENSHOT 1 — Home screen (hero shot) ───────────────
Screen: Home screen with 2–3 active tasks visible
Overlay headline:  "Remind me when I get there"
Overlay subtext:   "Not when the clock says so"
Placement: Bottom bar


── SCREENSHOT 2 — Create task ───────────────────────────
Screen: Create Task form, place picker open showing a saved place selected
Overlay headline:  "Pin a task to any place"
Overlay subtext:   "Store, gym, office — wherever you need to remember"
Placement: Bottom bar


── SCREENSHOT 3 — Notification moment ───────────────────
Screen: Android notification shade showing a Do In Place reminder notification
(or the home screen with a notification badge)
Overlay headline:  "You just arrived. We noticed."
Overlay subtext:   "Reminder fires the moment you're near"
Placement: Top bar


── SCREENSHOT 4 — Shopping list ─────────────────────────
Screen: Task detail of a shopping list task, items visible with checkboxes
Overlay headline:  "Shopping lists built for how stores work"
Overlay subtext:   "Check items off as you go"
Placement: Bottom bar


── SCREENSHOT 5 — Auto Order / learned sort ─────────────
Screen: Shopping list after tapping "Auto Order" — items reordered
(or SortList screen with "Save as Default for This Place" visible)
Overlay headline:  "It learns your store layout"
Overlay subtext:   "Items sort the way you actually walk"
Placement: Bottom bar


── SCREENSHOT 6 — Shared task / contacts ────────────────
Screen: Task detail showing a task assigned to a contact (assignee chip visible),
or Contacts screen showing a trusted contact
Overlay headline:  "Send tasks to people you trust"
Overlay subtext:   "They get reminded when they arrive"
Placement: Bottom bar


── SCREENSHOT 7 — Archive & restore ─────────────────────
Screen: Archived Tasks screen with 1–2 archived task cards visible, Restore button shown
Overlay headline:  "Nothing is ever gone for good"
Overlay subtext:   "Swipe to archive. Restore any time."
Placement: Top bar


── SCREENSHOT 8 — Priority + scheduling ─────────────────
Screen: Create Task or Task Detail showing Urgent priority chip and active date/time range
Overlay headline:  "Urgent or No Rush — you decide"
Overlay subtext:   "Reminders only fire when it makes sense"
Placement: Bottom bar


── EXPORT SPECS ─────────────────────────────────────────
Format:       PNG (no alpha required)
Min size:     1080 × 1920 px (portrait)
Colour space: sRGB
File size:    under 8 MB each
